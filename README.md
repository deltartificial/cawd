Terminal file explorer with syntax highlighting.

## Install

```bash
cargo install --path .
```

## Usage

```bash
cawd [path]
```

## Keybindings

| Key | Action |
|-----|--------|
| j/k | Navigate |
| h/l | Collapse/Expand |
| Enter | Open file |
| Ctrl+P | Search files |
| / | Filter/Search |
| Tab | Switch panel |
| q | Quit |
